# vue-picker
基于 vue 的移动端滑动选择组件，该仓库中包含了 `h-dialog` 弹出层组件、`h-picker` 滑动选择效果组件 以及 将前2者结合实现的 `h-slide-picker` 滑动弹层选择组件。
